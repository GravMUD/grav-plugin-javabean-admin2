Backend templating — 2004 called, it wants its pun back.

Free Admin2 theming for Grav 2.0 — twelve preset cockpits with paired light/dark tokens, live card picker, and Paint Shop controls. Rides Andy's appearance switcher; uninstall returns stock Admin2.

Paint Shop

Accent, density, typography, corners, custom CSS — power users without forking SvelteKit.

Requirements

  • Grav 2.0+
  • Admin2
  • API plugin

Install

bin/gpm install grav-javabean-admin2

Or extract to user/plugins/grav-javabean-admin2 and clear cache.
